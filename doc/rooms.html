handle_create/1

handle_create(Name) -> {ok, #room{}} | not_ok

  • Name = bitstring()

Handles the incoming _create room_ message with a specified Name. This will create a new room process if it does not yet exist, and add it to the tracking table.

handle_enter/1

handle_enter(Name) -> ok | not_ok

  • Name = bitstring()

Handles the incoming _enter room_ message. Return if a room is enterable, and if so, also returns the room object.

handle_getlist/1
